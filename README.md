# Smart-student-time-manager
An AI-powered intelligent time management system for students featuring adaptive learning, smart prioritization, and personalized daily planning. Built with Python/Streamlit, it automatically ranks tasks, generates optimized schedules, and learns from study patterns.

Smart Student Time Manager (MVP)
Prerequisites:
Python 3.9 or higher

Install required libraries:
bash
pip install streamlit pandas
Run the project:
bash
streamlit run app.py
